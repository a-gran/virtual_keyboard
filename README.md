# Программа отслеживания нажатий клавиш

## Описание
Программа в реальном времени отслеживает все нажатия клавиш на клавиатуре.

## Файлы

1. **keylogger.py** - полная версия
   - Показывает нажатия и отпускания клавиш
   - Отображает время каждого события
   - Подробный вывод информации

2. **keylogger_simple.py** - упрощенная версия
   - Показывает только нажатия клавиш
   - Минимальный вывод
   - Проще для понимания

3. **virtual_keyboard.py** - виртуальная клавиатура
   - Визуализация нажатий клавиш в реальном времени
   - Автоматическое переключение раскладки (RU/EN) синхронно с системной
   - Счетчик нажатий клавиш
   - Цветовая индикация текущей раскладки

## Как запустить

### Вариант 1: Полная версия

```bash
python keylogger.py
```

### Вариант 2: Упрощенная версия

```bash
python keylogger_simple.py
```

### Вариант 3: Виртуальная клавиатура

```bash
python virtual_keyboard.py
```

## Как остановить программу

Нажмите клавишу **ESC** на клавиатуре (для keylogger версий) или просто закройте окно виртуальной клавиатуры

## Требования
- Python 3.6 или новее
- Библиотека pynput (устанавливается автоматически)

## Примечание
⚠️ Использование программ отслеживания клавиш может регулироваться законодательством.
Используйте эту программу только на своем компьютере и для обучения.

## Возможности расширения

Вы можете модифицировать программу для:
- Сохранения нажатий в файл
- Подсчета статистики нажатий
- Фильтрации определенных клавиш
- Создания горячих клавиш
